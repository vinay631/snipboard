# SnipHarvest P1 Tasks

P1 tasks are high priority — targeted for Phase 2 & 3 after MVP ships.

---

## Organization Features

| ID | Task | Description |
|----|------|-------------|
| FR-4.1 | Tags System | Add/remove multiple tags per snippet with autocomplete from existing tags; filter by tag; support tag management (rename, delete, merge); show tag indicators on snippet cards |
| FR-4.2 | Favorites / Starred | Star/unstar toggle on snippet cards; dedicated filter view for favorites; star icon visible in list view |
| FR-4.3 | Sorting Options | Sort snippets by date (newest/oldest), page title (A-Z), or relevance; persist sort preference across sessions via dropdown in header |
| FR-4.4 | Folders / Collections | Create, rename, and delete folders; move snippets between folders; sidebar navigation for folders; each snippet belongs to one folder; default "Uncategorized" folder |

## Context Preservation

| ID | Task | Description |
|----|------|-------------|
| FR-5.1 | Context Extraction | Capture 500 characters before and after each selection, respecting paragraph boundaries; handle edge cases (snippet at page start/end); strip HTML tags but preserve line breaks |
| FR-5.2 | Context Display | Highlight the snippet within its surrounding context in the detail view; truncate long context with "…"; expandable/collapsible context section with clean formatting |

## Export & Backup

| ID | Task | Description |
|----|------|-------------|
| FR-6.1 | Export All Snippets | Export full collection as JSON, Markdown, or CSV; include all metadata; filename includes timestamp |
| FR-6.2 | Import Snippets | Import from previously exported JSON; validate file structure before importing; handle duplicates (skip or merge); show import progress and results summary |
