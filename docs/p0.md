# SnipHarvest P0 Tasks

P0 tasks are critical for MVP — the extension cannot ship without these.

---

## Text Selection & Capture

| ID | Task | Description |
|----|------|-------------|
| FR-1.1 | Text Selection Detection | Detect user text selection on any webpage; minimum 3 characters, works on SPAs and dynamically loaded content |
| FR-1.2 | Capture Action Trigger | Show floating capture button within 0.3s of selection, positioned near selected text; also expose right-click context menu option |
| FR-1.3 | Snippet Capture | Save selected text (up to 5,000 chars) with page URL, page title, timestamp, surrounding context (500 chars before/after), and a unique UUID; complete within 1 second |
| FR-1.4 | Capture Confirmation | Show a 2-second toast notification with snippet preview, a "View" link, and an undo option within 5 seconds |

## Storage & Data Management

| ID | Task | Description |
|----|------|-------------|
| FR-2.1 | Local Storage | Persist snippets via `chrome.storage.local`; support up to 10,000 snippets with retrieval under 100ms, surviving browser restarts |
| FR-2.2 | Data Structure | Implement the standardized snippet schema: required fields (`id`, `text`, `url`, `pageTitle`, `context`, `timestamp`, `createdDate`) plus optional fields (`tags`, `notes`, `color`, `isFavorite`, `folderId`) |
| FR-2.3 | Data Persistence | Auto-save on every capture; handle storage quota gracefully; ensure no data loss on browser crash |

## Popup Interface

| ID | Task | Description |
|----|------|-------------|
| FR-3.1 | Snippet List View | Display snippets as cards showing a 150-char preview, source favicon + truncated title, and relative capture date; paginate at 20 per page; show empty state for new users |
| FR-3.2 | Snippet Detail View | Show full snippet text, expandable context, clickable source URL, full metadata, and Copy / Edit / Delete actions |
| FR-3.3 | Search Functionality | Real-time search bar across snippet text, page titles, URLs, and tags; highlight matching terms; show "No results" message when nothing matches |
| FR-3.4 | Basic Actions | Support full CRUD: delete with confirmation, edit text (preserve original), copy to clipboard, open source URL in new tab — all completing within 1 second |
